- **📊 丰富的筛选功能**
  - 搜索插件名称和描述
  - 评分筛选（优秀/良好/一般/较差）
  - 开发者类型（交易者/个人）
  - 插件大小（轻量/小型/中型/大型）
  - **📅 更新时间**（近一周/近一个月/近三个月/近半年/近一年/一年以上）

- **💾 完整的插件信息**
  - 12,500+ 插件数据
  - 下载量、评分、评分人数
  - 开发者类型标识（99%覆盖率）
  - 插件大小（99%覆盖率）
  - 评分人数（72%覆盖率）
  - 支持语言（33%覆盖率）
